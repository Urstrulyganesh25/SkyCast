# SkyCast 🌤

SkyCast is a responsive weather web application that displays real-time weather data using the OpenWeather API.

## Features

- City weather search
- Current temperature
- Humidity and feels-like temperature
- Geolocation weather
- Mobile responsive UI
- And many more...

## Technologies

- HTML
- CSS
- JavaScript
- OpenWeather API
